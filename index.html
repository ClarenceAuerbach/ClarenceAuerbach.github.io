How to Play

  • Paddle (Memory Leak): Use LEFT/RIGHT arrow keys to control the memory leak and catch bits
  • Ball (Bits): Bouncing bits represent corrupted data
  • Bricks (Code): Break through C code snippets to earn points
  • Objectives: Break all code blocks before losing all lives
  • Danger: Let the bits fall through and you lose a life!

The Story

Your C program is under attack! Malicious bits are bouncing through your memory, corrupting code snippets left and right. As a memory leak, you must contain the damage by catching the bits before they escape. Break the corrupted code blocks to clean up your program and restore system stability!
